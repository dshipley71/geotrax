Media Extractor
===============
I. Create a credentials file in .aws directory containing the following
   information:

  [default]
  aws_access_key_id=[your key]
  aws_secret_access_key=[your secret key]
  region=us-east-1

  Sign up for an AWS account and obtain required keys.
  
  Save file as ~/.aws/credentials (no file extension)
  
  This will allow media extractor application to uploiad and download
  processed file results.
